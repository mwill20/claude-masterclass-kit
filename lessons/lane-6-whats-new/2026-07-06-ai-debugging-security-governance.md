## A. Index
Two disciplines in one week: systematic AI-assisted debugging (reproduce,
isolate, fix, verify, with the agent doing legwork under supervision) and
the governance frame for agentic tools: what may act, on what, with whose
approval, logged where.

## B. Build
Take one bug from your backlog. Run a full debugging loop with Claude Code
where every hypothesis gets a test before any fix lands. Separately, write
a half-page governance card for one agentic workflow you run: allowed
actions, forbidden actions, approval points, audit trail location.
You're done when: the bug has a verified fix with the failing-then-passing
test, and the governance card covers all four fields.
