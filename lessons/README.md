# Lessons

or to Lane 6 while it is still unmapped.

- `domain-1-agent-architecture/` - agent architecture and orchestration (4 stations)
- `domain-2-tool-design-mcp/` - tool design and MCP integration (2 stations)
- `domain-3-configuration-workflows/` - Claude Code configuration and workflows (7 stations)
- `domain-4-prompt-structured-output/` - prompt engineering and structured output (1 station)
- `domain-5-context-reliability/` - context management and reliability (2 stations)
- `lane-6-whats-new/` - staging for new capabilities that do not map cleanly yet (3 stations)

`seed-map.md` records which course document seeded which station. Station
counts above are as of 2026-07-10; the folders themselves are authoritative.

Mini assignments are not lesson source. They are examples of practice shape only.
